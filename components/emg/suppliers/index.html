    EMG 供应商大全¶

    肌电图 (EMG) 传感器模块、电极和专业设备供应商资源。

    消费级模块供应商¶

    DFRobot ⭐⭐⭐⭐⭐¶

    MVP 首选

    性价比高、文档齐全、社区活跃

    产品 通道 接口 价格 评级
    SEN0240 1 模拟 ¥319 ✅ 推荐
    SEN0241 1 I²C ¥359 数字输出

    产品规格 (SEN0240):

    • 输入阻抗: >10 MΩ
    • 带宽: 20-500 Hz
    • 增益: 1000x
    • 电源: 3.3V-5V
    • 输出: 0-3V 模拟

    联系方式:

    • 官网: dfrobot.com.cn
    • QQ 群: 技术交流群
    • Wiki: wiki.dfrobot.com.cn/SEN0240

    示例代码:

    #define EMG_PIN A0

void setup() {
  Serial.begin(115200);
}

void loop() {
  int emgValue = analogRead(EMG_PIN);
  Serial.println(emgValue);
  delay(1);  // 1 kHz
}
