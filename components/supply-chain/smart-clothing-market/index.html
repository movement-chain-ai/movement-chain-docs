    中国智能运动服装与可穿戴设备市场调研报告¶

    概述¶

    本报告专注于中国智能运动服装和可穿戴设备市场，涵盖智能运动服装、触觉反馈产品、智能鞋垫、智能手套、ODM/OEM服务商以及淘宝/1688现货产品等领域。旨在为开发者和初创企业提供完整的中国可穿戴生态系统供应商信息。

    1. 智能运动服装 (Smart Sports Clothing)¶

    1.1 米家运动心电T恤 (小米生态链)¶

    产品特点：

    • 60秒快速心电监测，每秒采样250次
    • 实时运动心率监测
    • 内置ADI心电芯片
    • 心电豆显示不同心率区间的颜色（热身、燃脂、增肌）

    价格：

    • 众筹价：249元
    • 正式售价：299元

    购买渠道：

    • 小米商城官网
    • 小米授权渠道

    材质：

    • 49% 锦纶、48% 聚酯纤维、3% 氨纶

    SDK/API：

    • 需通过小米IoT开发者平台（iot.mi.com）接入

    参考来源：

    • 科学健身神器？小米出了一款能监测心电数据的 T 恤
    • 米家运动心电T恤众筹 249 元

    1.2 棉捷科技 bodyta 纤维传感器¶

    公司背景：

    • 成立时间：2014年8月8日
    • 所属：小米生态链公司
    • 投资方：小米集团、顺为资本、鼎祥资本

    核心技术：

    • 纺织品传感器（Textile Sensor）
    • 弹性纤维电极
    • 纤维压力传感矩阵
    • 纤维拉力传感器

    应用领域：

    • 心电衣、智能床、人体工学椅
    • 宇航、军事、医疗

    主要合作伙伴：

    • 小米、华为、上汽、中国人寿、安踏
    • 中国军科院、中国探月计划、中国航天员训练中心

    联系方式：

    • 官网：www.bodyta.com
    • 深圳办公地址：深圳市南山区前海深港青年梦工场7-118
    • 北京：棉捷（北京）网络科技有限公司

    SDK/API：

    • 搜索结果未显示公开SDK，需直接联系咨询
    • 作为小米生态链，可能需通过小米IoT平台接入

    参考来源：

    • bodyta纤维压力传感器全球首发
    • 棉捷（深圳）网络科技有限公司招聘

    1.3 国际品牌在中国¶

    Hexoskin 智能运动背心：

    • 集成生物传感器
    • 每分钟收集42,000个数据点
    • 监测心率、心率变化/恢复、步数、卡路里、呼吸
    • 支持夜间睡眠追踪

    Athos 智能运动衣：

    • 内置EMG传感器（肌肉活动监测）
    • 心率、呼吸频率监测

    Ralph Lauren OMSignal 智能网球衫：

    • 内建心率、呼吸、心理压力传感器
    • 导电银线编织到布料中

    • 这些智能运动服饰能买到

    1.5 智能袜子¶

    MIT CSAIL 智能袜子研究：

    • 通过脚底压力监测姿态
    • 可识别蹲起、弓步、侧弓步、转体、侧抬腿、行走、上下台阶等动作

    中国市场趋势：

    • 内置传感器的运动袜可实时监测步态、压力分布
    • 预测2025年中国袜子行业规模突破千亿元
    • 智能袜等细分品类增速显著

    华为智能穿戴：

    • 玄玑感知系统涵盖60多项检测指标
    • 2024年Q1腕上可穿戴设备出货量全球第一

    医疗级智能袜子：

    • Siren Diabetic Socks：足底6个关键点温度监测
    • 提高87%的糖尿病足预防效率

    参考来源：

    • 把传感器穿身上！MIT发明智能袜子
    • 华为运动健康发布玄玑感知系统

    2. 触觉反馈产品 (Haptic Feedback Products)¶

    2.1 振动马达供应商¶

    深圳市普田振动电机有限公司¶

    公司背景：

    • 成立时间：2000年
    • 厂房面积：4.5万平方米
    • 年产量：超过50万台
    • 产品系列：28个系列，500多个型号

    主要产品：

    • MVE、VBE、YZS、YZO、YZU振动电机系列
    • PUTA系列铝合金振动电机

    技术参数（MVE系列）：

    • 电压：380V/660V三相（可定制）
    • 频率：50HZ/60HZ
    • 防护等级：IP65（防尘防水）
    • 工作温度：-20～+40℃
    • 激振力：三相 0～9098kg
    • 认证：CCC、CE认证

    联系方式：

    • 电话：18923719730 / 0755-27460501 / 18923423523
    • 地址：深圳市宝安区沙井万丰九八工业城
    • 官网：https://www.putiamotor.com/

    定制服务：

    • 支持小批量、多品种个性化定制
    • 价格和起订量需直接联系厂家咨询

    参考来源：

    • 深圳市普田振动电机有限公司

    日本电产科宝（Nidec）中国¶

    特点：

    • 响应速度灵敏
    • 触觉反馈体验丰富

    参考来源：

    • 振动马达 & 触觉装置

    2.2 EMS电刺激健身设备¶

    市场概况：

    • 中国市场约30余家EMS健身品牌
    • 大部分设备依赖进口（德国、韩国为主）
    • 国产自研设备相对较少

    主要中国EMS品牌：

    K-EMS（快马仕科技健身）¶

    • 成立时间：2016年
    • 门店数量：200余家（约90%加盟店）
    • 融资：数千万元Pre-A轮，估值3亿元
    • 设备来源：韩国

    ZESPEED（一普体育旗下）¶

    • 设备品牌：德国 Miha Bodytec
    • 2016年与Miha Bodytec签订协议，引入中国

    安体倍力（Fastall）¶

    • 融资：2018年天使轮1000万元
    • 设备品牌：德国 Amplitrain

    市场现状：

    • 2018年后市场收缩，多数品牌减少门店
    • 电商平台200-300元腹肌贴成为竞争对手
    • 设备成本较高（国际设备为主）

    国际设备制造商：

    • 德国 Miha Bodytec（全球最大EMS设备品牌）
    • 40多年运动与医疗器械历史
    • 12年专业运动设备研发经验

    参考来源：

    • 健身黑科技"EMS电脉冲"
    • 门店陆续关停，EMS电击健身风口不在

    2.3 触觉反馈背心¶

    bHaptics TactSuit¶

    产品特点：

    • 32个振动触觉电机
    • 无延迟无线连接
    • 超过250个SteamVR和Oculus Quest游戏原生支持
    • 音频转触觉反馈功能

    产品型号与价格：

    • TactSuit Pro（X40替代版）：32个振动点，500美元，13.5小时续航
    • TactSuit Air（X16替代版）：250美元

    购买途径：

    • 官网：www.bhaptics.com（支持支付宝）
    • 淘宝代理
    • 运费税费：13%起步

    兼容性：

    • 通过手机应用无线连接Quest、PC、PlayStation 5
    • 支持100+款Meta Quest游戏、200+款PC VR游戏、9款PSVR2游戏

    中国替代品：

    • 搜索未找到明确的中国国产替代品
    • 建议在淘宝、京东搜索"VR触觉背心"或"触觉反馈服装"

    参考来源：

    • 购买下一代全身触觉服 - bHaptics TactSuit
    • 效果最好的vr触感背心BHaptics TactSuit

    Teslasuit 全身触觉反馈服¶

    产品特点：

    • 全球首款全身触觉动捕服
    • 68个触觉点
    • 11个全身运动传感器支持动捕
    • 触觉反馈、运动捕捉、温度控制、生物识别

    13. 高尔夫可穿戴生态系统建议¶

    13.1 完整解决方案组合¶

    智能服装层：

    • 上衣：棉捷科技bodyta纤维传感器（心率、呼吸）
    • 手套：电子皮肤（福莱新材/赛感科技）- 压力感应
    • 袜子/鞋垫：力感科技柔性压力传感器 - 足底压力分析

    肌肉监测层：

    • EMG肌电传感器：思知瑞科技 或 唯理科技（手臂肌肉活动）

    触觉反馈层：

    • 振动马达模块：普田振动电机（姿势纠正反馈）
    • 可选：bHaptics触觉背心（训练反馈）

    13.2 开发路径建议¶

    阶段1：快速原型验证（1-2个月）¶

    1. 采购现货产品：
    2. 米家运动心电T恤（心率基础数据）
    3. 力感科技压力传感器（足底压力）

    4. 普田振动马达（触觉反馈原型）

    5. 开发：

    6. 使用小米IoT平台快速接入心率数据
    7. 自研或使用力感科技提供的接口采集压力数据
    8. 简单触觉反馈算法

    阶段2：定制开发（3-6个月）¶

    1. 与ODM厂商合作：
    2. 苏州能斯达：定制柔性传感器阵列（手套、袜子）
    3. 思知瑞科技：定制EMG采集方案

    4. 普田振动电机：定制振动马达布局

    5. 软件开发：

    6. 多传感器数据融合
    7. 高尔夫动作识别算法
    8. 实时反馈系统

    阶段3：批量生产（6个月+）¶

    1. 与电子皮肤厂商深度合作：
    2. 福莱新材或赛感科技：批量供应电子皮肤

    3. 苏州能斯达：批量柔性传感器

    4. ODM服务商：

    5. Thinkrace、多美达：整机组装
    6. 柔性传感器集成到服装
