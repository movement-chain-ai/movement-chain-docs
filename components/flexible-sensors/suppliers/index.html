    技术对接注意事项¶

    国内供应商¶

    1. 无 SDK 问题:
    2. 需要自行设计 MCU 采集电路
    3. 使用 ESP32 ADC 采集模拟信号

    4. 参考供应商提供的特性曲线

    5. 定制流程:

    6. 先小样验证
    7. 确认规格后批量

    8. 明确 MOQ 和交期

    9. 技术支持:

    10. 主要通过电话/微信
    11. 可能需要签 NDA

    国际供应商¶

    1. SDK 集成:
    2. 查看开发者文档
    3. 下载 SDK 示例代码

    4. BLE GATT 协议对接

    5. 采购流程:

    6. 官网直接下单
    7. 或联系销售代表
    8. 支持国际支付
