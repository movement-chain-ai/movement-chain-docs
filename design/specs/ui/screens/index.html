    屏幕列表¶

    文档目的: 定义应用的完整屏幕结构和导航架构

    目标读者: UI/UX 设计师、iOS 开发工程师

    1. 导航结构¶

    1.1 主导航 (Tab Bar)¶

    1.2 导航层级¶

    层级 说明 示例
    L1 主页面 Tab Bar 直达 首页、录制、历史、设置
    L2 功能页面 Push 进入 评估报告、练习详情、传感器配对
    L3 详情页面 Push 进入 问题详情、Drill 演示、历史对比
    Modal 模态弹出 录制预览、确认弹窗、选择器

    3. 屏幕状态定义¶

    3.1 通用状态¶

    状态 说明 视觉表现
    空态 (Empty) 无数据 插画 + 引导文案 + 操作按钮
    加载态 (Loading) 数据加载中 骨架屏 或 加载动画
    错误态 (Error) 加载失败 错误提示 + 重试按钮
    离线态 (Offline) 无网络 离线提示 + 可用功能
