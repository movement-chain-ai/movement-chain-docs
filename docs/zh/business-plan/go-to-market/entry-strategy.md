## 二、EMG 独特洞察

- 肌肉代偿检测 (核心弱 → 手臂代偿)
- 发力时序分析 (髋→核心→肩→臂 顺序)
- 疲劳预警 (在受伤前提醒休息)
- 左右不平衡 (识别弱侧)

```text
场景: 用户挥杆"感觉不对"

❌ 无 EMG:  视觉显示"转肩不足" → 用户不知道为什么
✅ 有 EMG:  检测到"核心激活弱 → 肩部代偿" → 给出针对性训练
```

!!! warning "EMG 是把双刃剑"

    | 维度 | 优势 | 劣势 |
    |------|------|------|
    | **功能** | 唯一能告诉"为什么"做错 | - |
    | **差异化** | 市场 0 家竞品有 | - |
    | **佩戴** | - | 多 2 个传感器，需贴皮肤 |
    | **使用门槛** | - | 电极片需正确放置，有学习成本 |
    | **维护** | - | 电极片是耗材，需定期更换 |
    | **价格** | - | 比纯 IMU 版贵 $150 |

    **策略：**

    1. **产品分层** - 纯 IMU 版 $399 入门，EMG 版 $549 给进阶用户
    2. **降低门槛** - App 提供电极放置引导视频
    3. **强调价值** - 不是"多了传感器"，而是"终于能看到为什么"
    4. **软件赋能** - 充分发挥 EMG 数据价值：高精度算法、直观可视化、友好交互界面
    5. **人性化设计** - 传感器贴合人体工学、快速穿戴、舒适材质

## 三、三种训练场景

我们针对不同场景设计了三种训练场景：

| 场景 | 说明 | 反馈方式 |
|------|------|----------|
| **Setup Check** | 站位准备时 | 视觉骨架叠加 + 语音提示 |
| **Slow Motion** | 慢动作练习 | **实时语音纠正** (创新点) |
| **Full Speed** | 正常挥杆后 | 挥杆后语音总结 |

!!! tip "创新: 慢动作模式"
    用户以慢速执行挥杆，系统可在动作过程中提供实时反馈。解决了"正常挥杆太快无法实时纠正"的难题。

> 详细技术规格见: [评估模式](../../design/specs/modes/assessment-mode.md)

## 四、进入策略

### 4.1 目标用户

| 用户类型 | 定位 | 策略 |
|---------|------|------|
| **A1 类玩家** | 核心利润 | 愿意持续付费、追求长期进步 → 重点服务，订阅/高价方案 |
| **A2 类玩家** | 可服务 | 一次性分析、偶尔复购 → 标准化产品，控制人工成本 |
| **A3 类玩家** | 不服务 | 只想免费、质疑多付费低 → 主动过滤 |
| **B 类教练** | 战略用户 | 口碑传播节点 → 让其"不反对"即可，不追求付费 |
| **C/D 类** | 暂不考虑 | 球场/俱乐部/专业运动员 → 未来机会 |

> 详见：[用户画像](../market-insights/user-personas.md)

### 4.2 SWOT 分析

| 优势 (S) | 劣势 (W) |
|----------|----------|
| EMG 差异化，市场唯一 | 品牌认知度低 |
| 三模态融合技术 | 资源有限 |
| 本土化优势 | 供应链待建立 |
| 开源社区支持 | 无成熟销售渠道 |

| 机会 (O) | 威胁 (T) |
|----------|----------|
| 中国市场空白 | 巨头可能进入 |
| 技术成熟可落地 | 价格战风险 |
| 智能手机普及 | 专利风险 |
| 健康意识提升 | 市场教育成本高 |

### 4.3 定位策略

| 维度 | 策略 |
|------|------|
| **产品** | 纯 IMU 版入门 + EMG 版进阶，满足不同用户需求 |
| **价格** | $399 / $549，与消费级同价但功能超越 |
| **市场** | 美国优先 → 全球扩展 |
| **渠道** | DTC 直销 + 教练合作推荐 |
| **定位** | "课后辅助工具"，不替代真人教练 |

### 4.4 Go-to-Market 路径

```text
Phase 1: 验证
├── MVP 开发完成
├── 早期用户测试 (50 人)
├── 产品快速迭代
└── 核心指标：用户留存 > 60%

Phase 2: 启动
├── 众筹验证 (Kickstarter / 淘宝众筹)
├── KOL 合作 (高尔夫博主、教练)
├── 建立教练合作伙伴网络 (10+ 家)
└── 核心指标：众筹目标达成率

Phase 3: 扩展
├── 正式销售渠道 (官网 DTC + 电商平台)
├── 区域扩展 (日韩、东南亚)
├── B2B 拓展 (高尔夫学校、球场)
└── 核心指标：年销量 > 1,000 台，实现盈利
```
