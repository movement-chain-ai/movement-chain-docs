## 国内 vs 国际供应商

### 国内供应商优势

| 优势 | 说明 |
|-----|------|
| **价格** | 通常便宜 30-70% |
| **物流** | 3-5 天到货 |
| **定制** | 小批量定制灵活 |
| **沟通** | 中文技术支持 |

### 国际供应商优势

| 优势 | 说明 |
|-----|------|
| **SDK** | 通常更完善 |
| **文档** | 英文文档齐全 |
| **认证** | FCC/CE 等国际认证 |
| **品质** | 一致性更好 |
