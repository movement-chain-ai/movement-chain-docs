## EMG 为什么是杀手级功能

```text
场景: 用户挥杆"感觉不对"

❌ 无 EMG:  视觉显示"转肩不足" → 用户不知道为什么
✅ 有 EMG:  检测到"核心激活弱 → 肩部代偿" → 给出针对性训练
```

**EMG 独特洞察**:

- 肌肉代偿检测 (核心弱 → 手臂代偿)
- 发力时序分析 (髋→核心→肩→臂 顺序)
- 疲劳预警 (在受伤前提醒休息)
- 左右不平衡 (识别弱侧)
