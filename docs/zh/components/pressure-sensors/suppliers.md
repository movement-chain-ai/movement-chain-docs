### DIY FSR 阵列 ⭐⭐⭐

| 方案 | 传感器数 | 成本 | 备注 |
|-----|---------|------|-----|
| FSR × 8 + ESP32 | 8 | ¥100-200 | MVP 方案 |
| FSR × 16 + MUX | 16 | ¥200-400 | 高分辨率 |

**实现方法**:
```cpp
// 8 点 FSR 阵列 - ESP32
#define NUM_FSR 8
const int fsrPins[] = {34, 35, 36, 39, 32, 33, 25, 26};

void readPressureArray(uint16_t* values) {
  for (int i = 0; i < NUM_FSR; i++) {
    values[i] = analogRead(fsrPins[i]);
  }
}
```

## 高尔夫应用注意事项

### 传感器布局

```text
                足底压力传感器布局 (8 点)

                    前掌 (Forefoot)
              ┌───────────────────────┐
              │   1       2       3   │
              │   ●       ●       ●   │
              │                       │
              │   4       5       6   │
              │   ●       ●       ●   │
              │                       │
              │       7       8       │
              │       ●       ●       │
              └───────────────────────┘
                    后跟 (Heel)

        关键监测区域:
        - 前掌内侧 (1, 4): 重心前移
        - 前掌外侧 (3, 6): 侧向稳定
        - 后跟 (7, 8): 上杆时后移
```
