## 技术对接注意事项

### 国内供应商

1. **无 SDK 问题**:
   - 需要自行设计 MCU 采集电路
   - 使用 ESP32 ADC 采集模拟信号
   - 参考供应商提供的特性曲线

2. **定制流程**:
   - 先小样验证
   - 确认规格后批量
   - 明确 MOQ 和交期

3. **技术支持**:
   - 主要通过电话/微信
   - 可能需要签 NDA

### 国际供应商

1. **SDK 集成**:
   - 查看开发者文档
   - 下载 SDK 示例代码
   - BLE GATT 协议对接

2. **采购流程**:
   - 官网直接下单
   - 或联系销售代表
   - 支持国际支付
