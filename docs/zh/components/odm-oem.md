## 合作流程

### ODM 合作流程

```text
1. 需求沟通    →    2. 方案评估    →    3. 样品开发
   (1-2 周)           (1 周)            (2-4 周)
                                            │
                                            ▼
6. 量产交付    ←    5. 试产验证    ←    4. 样品确认
   (4-8 周)           (2-4 周)            (1-2 周)
```

### 关键节点

| 节点 | 内容 | 注意事项 |
|-----|------|---------|
| **需求沟通** | 功能/性能/成本 | 明确核心需求 |
| **方案评估** | 技术可行性 | 确认 SDK 能力 |
| **样品开发** | 原型制作 | 预留调试时间 |
| **试产验证** | 小批量测试 | 质量把控 |
| **量产交付** | 批量生产 | 进度跟踪 |
