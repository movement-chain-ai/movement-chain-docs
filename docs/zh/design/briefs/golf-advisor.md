## 一、我们在做什么

Movement Chain AI 是一个结合 **IMU 传感器 + 肌电图 (EMG) + 视觉分析** 的高尔夫挥杆分析系统，旨在帮助业余球手：

1. **理解问题根源**：不只告诉你"手腕角度错了"，还能告诉你"因为核心没发力"
2. **练习场→球场闭环**：验证练习是否真的转化为球场成绩
3. **实时反馈**：击球后立即语音提示（无需盯着手机看数据）

### 我们与市场上其他产品的区别

| 产品类型 | 数据来源 | 能告诉你什么 | 不能告诉你什么 |
|---------|---------|-------------|--------------|
| **HackMotion** | 手腕 IMU | 手腕角度错了 | **为什么错**（缺肌肉数据） |
| **Sportsbox AI** | 摄像头 | 身体姿势问题 | 肌肉激活时序 |
| **Arccos** | GPS + 统计 | 球场表现数据 | 挥杆技术问题 |
| **Movement Chain AI** | IMU + EMG + Vision | 问题 + 根因 + 改进验证 | （这是我们的目标） |

### 1️⃣ 8 阶段划分是否合理？

我们采用 GolfDB 标准的 8 阶段划分：

```text
┌────────────────────────────────────────────────────────────┐
│                  Golf Swing 8 Phases                       │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  1. Address              站姿准备                          │
│  2. Toe-Up               杆头起                            │
│  3. Mid-Backswing        上杆中段                          │
│  4. Top                  上杆顶点                          │
│  5. Mid-Downswing        下杆中段（最大加速区）            │
│  6. Impact               击球瞬间                          │
│  7. Mid-Follow-Through   收杆中段                          │
│  8. Finish               收杆完成                          │
│                                                            │
└────────────────────────────────────────────────────────────┘
```

**我们想请教您：**

- ✅ **这个划分对实际教学有意义吗？**

  - 有没有更好的划分方式？

- ✅ **哪些阶段是关键检查点？**
  - 我们目前认为 Top 和 Impact 是最重要的，您同意吗？
  - 您觉得哪些阶段容易被忽视但其实很重要？

- ✅ **业余球手的典型问题出现在哪些阶段？**
  - 例如：早释放 (Early Release) 通常在 Phase 5 (下杆中段) 就能看出

### 2️⃣ 检测规则是否准确？

我们的 MVP 系统会检测以下 6 个核心问题：

| 规则 | 检测条件 | 阈值 | 反馈语言 |
|-----|---------|-----|---------|
| **倒序运动链** | 前臂先于核心激活 (EMG 时序差 < -20ms) | 严重问题 | "手臂比身体先动了，试着从核心启动下杆" |
| **过度手臂挥杆** | 前臂激活 / 核心激活比值 | > 1.3 | "感觉核心在发力，别只用手臂打" |
| **差异角过小** | X-Factor (肩髋角度差) | < 20° | "肩膀再多转一点，跟髋部拉开角度" |
| **节奏过快** | 下杆时长 | < 0.20s | "上杆再慢一点，别急着打" |
| **节奏过慢** | 下杆时长 | > 0.40s | "下杆可以更果断一些" |
| **早释放** | 手腕释放点 | < 40% 下杆完成 | "保持手腕角度，别太早释放" |

**我们想请教您：**

- ✅ **阈值合理吗？**
  - 例如 X-Factor < 20° 算严重问题，您同意吗？
  - 下杆时长 0.20-0.40s 是否符合您的经验？

- ✅ **反馈语言自然吗？**
  - 是否像您平时会说的话？
  - 怎样说会更鼓励球手、更容易理解？

- ✅ **有遗漏的重要规则吗？**
  - 您认为业余球手还有哪些高频问题我们没涵盖？
  - 例如：站姿问题、重心转移、髋部打开不够等

---

### 3️⃣ EMG (肌肉激活) 数据的教学价值

我们的独特优势是 EMG 传感器，可以检测：

| EMG 能检测的 | 具体指标 | 教学意义 |
|-------------|---------|---------|
| **核心肌群激活时序** | 核心 vs 前臂启动时间差 | 验证是否"从核心启动" |
| **发力比例** | 前臂/核心激活比值 | 识别"手臂挥杆"问题 |
| **疲劳检测** | 肌肉激活下降趋势 | 提醒休息，避免受伤 |

**我们想请教您：**

- ✅ **这些数据对教学有帮助吗？**
  - 您觉得最有价值的是哪一项？
  - 有没有您想看到但我们还没做的肌肉数据？

- ✅ **您会怎么用这个数据？**
  - 例如：用来证明"你确实在用手臂打，不是感觉问题"
  - 或者：用来验证改进效果（核心激活比例提升）

- ✅ **EMG 数据如何呈现更直观？**
  - 百分比？颜色编码？时序图？
  - 您希望看到实时数据还是总结报告？

- 鼓励为主，不要打击信心
- 简洁易懂，一句话说清
- 适合语音播报（无需看屏幕）

### 3. 帮我们录制"标准挥杆"视频

我们需要采集一些标准挥杆作为参考数据：

- **需求**：3-5 个不同水平的标准挥杆
  - 职业/教练水平 (作为"最佳示范")
  - 低差点球手 (10 差点以内)
  - 业余球手 (15-20 差点)

- **拍摄要求**：
  - 侧面 60fps 视频
  - 佩戴我们的 IMU + EMG 传感器
  - 打 5-10 球，我们会选取最好的 3 个

- **时间**：约 30 分钟 / 人

我们希望找到 3-5 位愿意参与早期测试的球友：
