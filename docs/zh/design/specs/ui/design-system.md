# 设计系统

> **文档目的**: 定义应用的视觉设计规范和组件库
>
> **目标读者**: UI/UX 设计师、iOS 开发工程师

---

## 1. 设计原则

### 1.1 核心原则

| 原则 | 说明 | 体现 |
|-----|------|------|
| **专业感** | 体现运动科学的专业性 | 数据可视化精确、术语规范 |
| **清晰** | 信息层次分明 | 卡片式布局、渐进式披露 |
| **高效** | 减少操作步骤 | 快速入口、智能默认值 |
| **激励** | 正向反馈 | 进步可视化、成就系统 |

### 1.2 设计语言

- **风格**: 现代运动科技感，参考 Apple Fitness+、Nike Training Club
- **氛围**: 专业但不冷冰冰，科技但保持亲和力
- **目标用户**: 30-50 岁高尔夫爱好者，注重品质感

## 2. 色彩系统

### 2.1 品牌色

| 颜色名称 | Hex | RGB | 用途 |
|---------|-----|-----|------|
| **Primary** (运动蓝) | `#0A84FF` | 10, 132, 255 | 主按钮、链接、强调 |
| **Primary Dark** | `#0066CC` | 0, 102, 204 | 按钮按压态 |
| **Primary Light** | `#B3D9FF` | 179, 217, 255 | 背景色、选中态 |

### 2.2 语义色

| 颜色名称 | Hex | 用途 |
|---------|-----|------|
| **Success** (绿色) | `#30D158` | 正确、完成、表现良好 |
| **Warning** (黄色) | `#FFD60A` | 警告、中等问题 |
| **Error** (红色) | `#FF453A` | 错误、严重问题、高优先级 |
| **Info** (蓝色) | `#64D2FF` | 提示、信息 |

### 2.3 问题严重程度色彩

| 级别 | 颜色 | Hex | 应用场景 |
|-----|------|-----|---------|
| **高优先级 (P0)** | 红色 | `#FF453A` | 发力顺序错误等核心问题 |
| **中优先级 (P1)** | 橙色 | `#FF9F0A` | X-Factor 不足等改进项 |
| **低优先级 (P2)** | 黄色 | `#FFD60A` | 站距偏窄等小问题 |
| **正面反馈** | 绿色 | `#30D158` | 表现良好的方面 |

### 2.4 中性色

| 颜色名称 | Hex | 用途 |
|---------|-----|------|
| **Background** | `#000000` | 主背景 (Dark Mode) |
| **Surface** | `#1C1C1E` | 卡片背景 |
| **Surface Elevated** | `#2C2C2E` | 弹窗背景 |
| **Border** | `#38383A` | 分割线 |
| **Text Primary** | `#FFFFFF` | 主文字 |
| **Text Secondary** | `#8E8E93` | 次要文字 |
| **Text Tertiary** | `#636366` | 辅助文字 |

### 2.5 浅色模式

| 颜色名称 | Hex | 用途 |
|---------|-----|------|
| **Background** | `#F2F2F7` | 主背景 |
| **Surface** | `#FFFFFF` | 卡片背景 |
| **Text Primary** | `#000000` | 主文字 |
| **Text Secondary** | `#8E8E93` | 次要文字 |

## 3. 字体系统

### 3.1 字体选择

| 平台 | 字体 | 备注 |
|-----|------|------|
| **iOS** | SF Pro | 系统字体，支持动态字体 |
| **数字** | SF Pro Rounded | 评分、计数等数字 |
| **代码** | SF Mono | 技术数据展示 |

### 3.2 字体层级

| 层级 | 名称 | 大小 | 字重 | 行高 | 用途 |
|-----|------|------|------|------|------|
| **Display** | 大标题 | 34pt | Bold | 41pt | 评分数字、大标题 |
| **Title 1** | 标题 1 | 28pt | Bold | 34pt | 页面标题 |
| **Title 2** | 标题 2 | 22pt | Bold | 28pt | 卡片标题 |
| **Title 3** | 标题 3 | 20pt | Semibold | 25pt | 列表标题 |
| **Headline** | 小标题 | 17pt | Semibold | 22pt | 列表项标题 |
| **Body** | 正文 | 17pt | Regular | 22pt | 段落正文 |
| **Callout** | 标注 | 16pt | Regular | 21pt | 表单标签 |
| **Subhead** | 副标题 | 15pt | Regular | 20pt | 辅助说明 |
| **Footnote** | 脚注 | 13pt | Regular | 18pt | 次要信息 |
| **Caption 1** | 小字 1 | 12pt | Regular | 16pt | 时间戳 |
| **Caption 2** | 小字 2 | 11pt | Regular | 13pt | 标签 |

### 3.3 使用示例

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   72                           ← Display (34pt Bold) 评分                   │
│   ───                                                                       │
│   / 100                        ← Title 3 (20pt Regular) 辅助                │
│                                                                             │
│   评估报告                      ← Title 1 (28pt Bold) 页面标题               │
│                                                                             │
│   需要改进                      ← Title 2 (22pt Bold) 卡片标题               │
│                                                                             │
│   发力顺序错误                  ← Headline (17pt Semibold) 列表项标题         │
│   下杆时手臂先于核心启动         ← Body (17pt Regular) 正文描述               │
│                                                                             │
│   2026-01-08 14:30             ← Caption 1 (12pt Regular) 时间戳            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 4. 间距系统

- **基础单位**: 4pt

### 4.2 间距规范

|-----|------|------|
| **xs** | 4pt | 紧凑元素间距 |

| **m** | 16pt | 标准间距 |

| **xl** | 32pt | 区块间距 |
| **xxl** | 48pt | 页面边距 (顶部) |

### 4.3 布局边距

| 位置 | 间距 | 说明 |
|-----|------|------|
| **页面水平边距** | 16pt | 内容与屏幕边缘 |
| **卡片内边距** | 16pt | 卡片内容与边框 |
| **卡片间距** | 12pt | 卡片之间的间距 |
| **Tab Bar 上方** | 34pt | 内容与 Tab Bar |

## 8. 动画

### 8.1 时长规范

| 类型 | 时长 | 用途 |
|-----|------|------|
| **快速** | 100ms | 按钮点击反馈 |
| **标准** | 250ms | 页面转场、卡片展开 |
| **慢速** | 400ms | 模态弹出、复杂动画 |

### 8.2 缓动曲线

| 类型 | 曲线 | 用途 |
|-----|------|------|
| **标准** | easeInOut | 大多数场景 |
| **弹性** | spring | 成功反馈、强调 |
| **线性** | linear | 进度条 |

### 8.3 常用动画

| 场景 | 动画类型 | 说明 |
|-----|---------|------|
| 页面转场 | Push/Pop | 从右侧进入/退出 |
| 模态弹出 | Slide Up | 从底部滑入 |
| 加载中 | Pulse | 骨骼屏闪烁 |
| 成功反馈 | Scale + Checkmark | 放大 + 勾选动画 |
| 数字变化 | Counter | 数字滚动 |
