### 2.3 四种功能模式

#### 评估模式 (Assessment Mode)

| 屏幕 ID | 文件 | 屏幕名称 | 功能描述 |
|--------|------|---------|---------|
| `assessment` | `AssessmentMode.tsx` | 评估模式 | 评估入口、类型选择 |
| `assessment/ready` | 内部状态 | 准备开始 | 传感器检查 |
| `assessment/intro` | 内部状态 | 选择类型 | 上杆/下杆/全挥杆 |
| `assessment/recording` | 内部状态 | 录制中 | 挥杆录制 |
| `assessment/report` | `AssessmentReport.tsx` | 评估报告 | 得分、问题列表、推荐练习 |

**评估流程**:

```text
准备开始 (ready) → 选择类型 (intro) → 传感器检查 → 录制 → 评估报告
```

**评估类型**:

- 上杆评估 (Backswing Assessment)
- 下杆评估 (Downswing Assessment)
- 全挥杆评估 (Full Swing Assessment)

#### 练习模式 (Drill Mode)

| 屏幕 ID | 文件 | 屏幕名称 | 功能描述 |
|--------|------|---------|---------|
| `drill` | `DrillMode.tsx` | 练习模式 | 练习库、训练计划 |
| `drill/detail` | `DrillDetail.tsx` | 练习详情 | 视频对比、肌肉数据 |
| `drill/plan` | `TrainingPlan.tsx` | 训练计划 | 计划列表、进度 |
| `drill/plan-detail` | `PlanDetail.tsx` | 计划详情 | 计划内容 |
| `drill/executor` | `PlanExecutor.tsx` | 传感器检查 | 训练前检查 |
| `drill/all` | `AllDrillsMode.tsx` | 训练执行 | 进度指示、执行 |

**两种入口流程**:

```text
练习库入口:   点击练习 → DrillDetail（无传感器检查）
训练计划入口: 点击"开始训练" → PlanExecutor传感器检查 → AllDrillsMode执行
```

**DrillDetail 功能**:

- 视图模式：上下分屏、画中画、叠加对比
- 肌肉部位：核心图解、侧面图、前面、三角肌、臀部
- 实时肌肉激活数据
- 倍速播放、录制功能

#### 引导模式 (Guide Mode)

| 屏幕 ID | 文件 | 屏幕名称 | 功能描述 |
|--------|------|---------|---------|
| `guide` | `GuideMode.tsx` | 引导模式 | 引导入口、类型选择 |
| `guide/ready` | 内部状态 | 准备开始 | 传感器检查 |
| `guide/intro` | 内部状态 | 选择类型 | 站姿/慢动作/整体 |
| `guide/practice` | `GuidedPractice.tsx` | 引导练习 | 实时引导界面 |

**引导类型**:

- 站姿引导
- 慢动作引导
- 整体引导

#### 实战模式 (Real Play Mode)

## 4. 屏幕状态定义

### 4.1 通用状态

| 状态 | 说明 | 视觉表现 |
|-----|------|---------|
| **空态 (Empty)** | 无数据 | 插画 + 引导文案 + 操作按钮 |
| **加载态 (Loading)** | 数据加载中 | 骨架屏 或 加载动画 |
| **错误态 (Error)** | 加载失败 | 错误提示 + 重试按钮 |
| **传感器断开** | 连接中断 | 警告提示 + 重连按钮 |

```typescript
type Page =
  | 'home'              // 首页

  | 'sensor'            // 传感器
  | 'swing-phase-detail'// 挥杆阶段详情

```

### 5.2 传感器数据接口

```typescript
interface Sensor {
  id: string;
  name: string;
  type: 'EMG' | 'IMU';
  status: 'connected' | 'disconnected';
  battery: number;
  signal: number;
  position?: string;
}
```
