# 传感器位置验证协议

> 基于学术研究的高尔夫挥杆动力链验证方案

## 概述

本协议使用 **3 个 IMU + 3 个 EMG** 传感器验证高尔夫挥杆的生物力学动力链 (Kinetic Chain)。

所有位置选择均基于同行评审的学术研究。

## Trail vs Lead 说明

```text
右撇子站位 (面向目标):

        目标方向 →

    ┌─────────────────┐
    │                 │
    │   左脚    右脚  │
    │   Lead   Trail  │
    │   前腿    后腿  │
    │                 │
    └─────────────────┘

Trail side = 离目标更远的一侧 = 动力启动侧
Lead side = 离目标更近的一侧 = 动力接收侧
```

**为什么贴 Trail side？**

- 研究显示 Trail leg 臀大肌激活与差点显著相关
- 低差点选手 Trail 侧臀大肌激活显著高于高差点选手
- 动力链从 Trail hip 启动，向 Lead side 传递

## 动力链验证目标

### 正确的动力链顺序

```text
Proximal → Distal (近端到远端)

骨盆 (Pelvis) → 躯干 (Thorax) → 手臂 (Arms) → 球杆 (Club)
     ↓              ↓              ↓            ↓
   477°/s        552°/s        1200°/s      2500°/s
   (先达峰)      (后达峰)       (更后)        (最后)
```

### IMU 验证指标

| 指标 | 正确模式 | 错误模式 |
|------|----------|----------|
| 峰值顺序 | Pelvis → Thorax → Hand | 顺序颠倒 |
| 骨盆-躯干时间差 | 20-50ms | <10ms 或 >100ms |
| X-Factor (Top位置) | 35-55° | <25° |

### EMG 验证指标

| 指标 | 正确模式 | 错误模式 |
|------|----------|----------|
| 激活顺序 | 臀大肌 → 竖脊肌 → 腕屈肌 | 腕屈肌先激活 ("手臂甩杆") |
| 臀-前臂时间差 | 臀大肌领先 30-60ms | 同时或颠倒 |
| 信号质量 | 激活 RMS > 3× 基线 | 信号弱或噪声大 |

## 肌肉选择的科学依据

### 为什么选这3块肌肉？

| 选择 | 原因 | 为什么不选其他 |
|------|------|----------------|
| **竖脊肌** | 躯干激活最高 (83-106% MVC) | ❌ 腹斜肌：研究显示"激活程度中等到**低**" |
| **腕屈肌 (FCR)** | 激活 >100% MVC，与击球效果直接相关 | ✅ 前臂最佳选择 |
| **臀大肌** | 髋部旋转的主驱动力 | ❌ 腘绳肌：激活高但主要是膝盖稳定作用 |

### 大腿 vs 臀部对比

| 肌肉 | 位置 | 激活水平 | 动力链角色 |
|------|------|---------|-----------|
| **臀大肌** | 臀部 | ⭐⭐ 中高 | **主驱动力** - 髋部旋转发动机 |
| 腘绳肌 | 大腿后侧 | ⭐⭐⭐ 70-76% (最高) | 膝盖稳定 + 辅助髋伸展 |
| 股四头肌 | 大腿前侧 | ⭐⭐ 中 | 重心转移、支撑 |

**结论**：腘绳肌激活最高，但主要是**稳定**作用；臀大肌才是**旋转驱动力**。

## 预期数据模式

### 正确的动力链 (好的挥杆)

```text
角速度时间线:

骨盆 (L4)  ───────╱╲─────────────────
                 ↑ 第1个达峰

躯干 (T1)  ────────────╱╲────────────
                      ↑ 第2个达峰 (晚20-50ms)

手部       ──────────────────╱╲──────
                            ↑ 第3个达峰 (最晚)

           ├────────────────────────→ 时间
          Top                    Impact
```

### 错误模式 (手臂甩杆)

```text
角速度时间线:

手部       ──────╱╲──────────────────
                ↑ 峰值太早

躯干 (T1)  ────────╱╲────────────────

骨盆 (L4)  ──────────╱╲──────────────
                    ↑ 峰值太晚 (应该最先)

           ├────────────────────────→ 时间
```
