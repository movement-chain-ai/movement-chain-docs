## 四、采集汇总

### 错误动作分配（对应6条诊断规则）

| 优先级 | 错误类型 | 触发条件 | 数据源 | 次数/人 | 总次数 | 动作指导 |
|-------|---------|---------|--------|--------|--------|---------|
| P0 | 倒序运动链 | 前臂先于核心激活 (gap < -20ms) | EMG | 2次 | 10次 | 刻意用手臂启动下杆 |
| P0 | 过度手臂挥杆 | Forearm/Core ratio > 1.3 | EMG | 2次 | 10次 | 核心不发力，纯手臂挥 |
| P1 | X-Factor 不足 | X-Factor < 20° | Vision | 1次 | 5次 | 最小化肩髋分离 |
| P1 | 节奏过快 | Downswing < 0.20s | IMU | 2次 | 10次 | 尽可能快地下杆 |
| P1 | 节奏过慢 | Downswing > 0.40s | IMU | 1次 | 5次 | 刻意放慢下杆速度 |
| P1 | 早释放 | Wrist release < 40% downswing | IMU | 2次 | 10次 | 下杆初期就打开手腕 |

## 五、验证目标

- [ ] IMU 检测 Top/Impact 准确率 >85%
- [ ] EMG 区分核心 vs 手臂顺序准确率 >80%
- [ ] 时间同步 <1ms
- [ ] 至少4条诊断规则可触发
- [ ] 不同水平球手指标差异可见
